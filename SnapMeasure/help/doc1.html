Main Page

This page provides access to the main functionalities of the application: project management, outcrop interpretation and export, picture management, features histogram and map view.

Project Management

Outcrop pictures and their interpretation are managed inside projects. A default project (Project 1) exists. The current project is used to filter outcrops for maps and features histogram.
Create a new empty project
Select a project to become the current project
Delete the current project with its attached images and interpretations

Outcrop Interpretation

Select an existing outcrop interpretation and resume editing. See next page for all the editing functionalities.

Picture

Take a new picture and interpret it. Location information, angle of view will be used to locate the interpretation in 3D for export.
Select a picture from your picture library and interpret it. The image may be missing location and orientation information.

Inventory

Compute an histogram of interpreted features (channels, lobes, etc.) heights and width inside the current project.
Shows the outcrops of the current project on a map.
